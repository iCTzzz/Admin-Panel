# CTz Management System

CTz Management System es una aplicación web tipo dashboard desarrollada como proyecto personal con fines demostrativos. Permite gestionar y visualizar información de clientes, proyectos y servicios mediante una interfaz moderna, clara y funcional.

---

## Descripción

Este proyecto simula un sistema administrativo utilizado en entornos empresariales, específicamente para la empresa MBS. Está diseñado para demostrar habilidades en desarrollo frontend, organización de código, diseño de interfaces y manejo de datos.

El sistema permite:

- Visualizar métricas clave del negocio
- Consultar información de clientes y proyectos
- Filtrar y buscar datos en tiempo real
- Navegar entre diferentes módulos de manera intuitiva

---

## Aviso importante

Este sistema es un proyecto personal desarrollado con fines demostrativos.  
La empresa "MBS" es una entidad ficticia creada únicamente para representar un entorno empresarial.

Todos los datos mostrados (incluyendo clientes, proyectos, ingresos y métricas) son simulados y no corresponden a información real de ninguna empresa o persona.

---

## Tecnologías

- React
- Vite
- Tailwind CSS
- JavaScript / TypeScript
- Recharts (visualización de datos)
- React Hooks
