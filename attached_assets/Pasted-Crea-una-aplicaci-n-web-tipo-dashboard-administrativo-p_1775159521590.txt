Crea una aplicación web tipo dashboard administrativo profesional para "MBS", una empresa que gestiona clientes, proyectos y servicios. El sistema debe permitir consultar, visualizar y filtrar información desde una base de datos (simulada o real), con una interfaz moderna, clara y altamente usable.

El proyecto debe reflejar buenas prácticas de desarrollo, diseño limpio y estructura escalable, ya que será utilizado como prueba técnica para evaluación profesional.

🧠 OBJETIVO DEL SISTEMA

Desarrollar un panel donde el usuario pueda:

Consultar registros desde una base de datos
Buscar información en tiempo real
Filtrar datos dinámicamente
Visualizar métricas clave del negocio
Navegar entre módulos de forma intuitiva
